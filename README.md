# Quiz Pariksha 🎓

A modern ed-tech platform connecting students and institutes. Built with **React**, **Vite**, and **Chakra UI v3**.

## 🛠 Tech Stack

* **Core:** React 18, Vite
* **UI/Styling:** Chakra UI v3, Lucide React, Next-themes
* **Routing:** React Router
* **State:** React Hooks


## 🚀 Quick Start

1. **Clone & Install**
```bash
git clone https://github.com/your-username/quiz-pariksha.git
cd quiz-pariksha
npm install

```



2. **Run Development Server**
```bash
npm run dev

```


Open `http://localhost:5173` to view the app.

## 📦 Scripts

* `npm run dev`: Start the dev server with HMR.
* `npm run build`: Build for production to the `dist` folder.
* `npm run preview`: Locally preview the production build.
* `npm run lint`: Run ESLint.

## 📂 Project Structure

```text
src/
├── components/   # Reusable UI components (Navbar, Cards)
├── pages/        # Route components (Home, Login, Signup, Features)
├── theme/        # Chakra UI configuration & token overrides
├── utils/        # Hooks (useAuth) and helpers
└── App.jsx       # Main entry & Routing logic

```
